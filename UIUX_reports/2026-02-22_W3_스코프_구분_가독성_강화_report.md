## 1) 작업 개요
홈/매치업 화면에서 전국/지역/기초 지표의 구분이 약해 직접 비교 오해 가능성이 있었다.  
W3에서는 스코프 라벨 문구와 톤을 통일하고, 비전국 스코프일 때 경고 카피를 명시해 비교 기준을 선명하게 고정했다.

## 2) 반영 내용
1. 스코프 공통 포맷 함수 정리(`apps/web/lib/format/scope_completeness.ts`)
- `scopeDisplayLabel` 추가
- `national -> 전국 지표`, `regional -> 지역(광역) 지표`, `local -> 기초(시군구) 지표`로 고정
- `scopeTone` 추가
- `national -> ok`, `regional -> neutral`, `local -> warn`으로 고정

2. 홈 요약 카드 가독성 보강(`apps/web/components/home/NationalSummaryCards.tsx`)
- 정당/대통령 카드의 스코프 배지에 `scopeDisplayLabel + scopeTone` 적용
- 비전국 스코프일 때 비교 주의 문구 추가

3. 매치업 상세 비교 안내 보강(`apps/web/app/matchups/[matchup_id]/page.tsx`)
- 스코프 배지 문구를 `scopeDisplayLabel`로 통일
- 전국/비전국 비교 가이드 배지를 분리 표시
- 본문 안내문에서 비전국일 때 `동일 스코프 비교 권장` 경고를 고정
